# File Manager Project

## Run
To run this project is very simple. Navigate to the directory with `file_manager.py` and run the command:
```
python3 file_manager.py
```

This will then open the file manager in the background. You can then navigate to it. 

### Dependencies
This program only contains dependencies from the Python Standard Library, meaning you don't have to install any through `pip`.

### Disclaimer
This project was only tested on MacOS Sequoia with Python 3.12.6. However, it should work with other operating systems and versions of Python as well.
